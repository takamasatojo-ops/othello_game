# othello game

オセロゲームをコマンド上で実行するためのスクリプト群です。

## ディレクトリ構成

```text
reversi/
  main.py           プログラム実行
  view.py           盤面や文字を表示
  model.py　　　　　　オセロゲームの処理を行う
  controller.py　　　全体操作、viewとmodelの橋渡し
```

## 遊び方

まず、コマンド上でreversiディレクトリに移動後、以下のコマンドを実行します。

```bash
python main.py
```

実行後に以下の画面が表示されます。

```text
  0 1 2 3 4 5 6 7
0 ･ ･ ･ ･ ･ ･ ･ ･
1 ･ ･ ･ ･ ･ ･ ･ ･
2 ･ ･ ･ ･ * ･ ･ ･
3 ･ ･ ･ B W * ･ ･
4 ･ ･ * W B ･ ･ ･
5 ･ ･ ･ * ･ ･ ･ ･
6 ･ ･ ･ ･ ･ ･ ･ ･
7 ･ ･ ･ ･ ･ ･ ･ ･
current turn:B
you can put your stone on {(4, 2), (5, 3), (2, 4), (3, 5)}
You put your stone on:
```
8x8の盤面が表示され、盤面の行と列番号が左側、上側にそれぞれ表示されます。それぞれ0〜7の数字が割り当てられています。
また、盤面の下には、現在のターン、そのターンの人が石を置けるマスの位置（相手の石をひっくり返す事ができるマスの位置）が表示されます。
盤面の表記については、以下の通りです。

```text
･:石が置かれていないマス
B:黒石
W:白石
*:現在のターンの人が石を置くことができるマス
```

まず、黒のターンの人が、石を置けるマスの位置の一覧から、石を置くマスを選んだら、「You put your stone on:」のところに、選んだマスの位置を入力します。
（一覧にないマスや不適切な値を入力すると、「Please input correct answer」と表示されて、入力のやり直しとなります。

正しい位置に石を置いたら、画面が切り替わり、石を置いた後の盤面の状態が表示されます。
（自分の石を置いたマスや相手の石をひっくり返したマスが自分の石に変わります）

盤面の下には、現在のターンが相手のターン（W）に切り替わり、次に、白のターンの人が石を置けるマスの位置が表示されます。
また、盤面も、白のターンの人が石を置くことができるマスが「*」で表示されます。

そして、白のターンの人が、石を置けるマスの位置の一覧から、石を置くマスを選んで入力する。
その後また、黒のターンの人が、石を置けるマスの位置の一覧から、石を置くマスを選んで入力する。

この繰り返しです。

## その他の実装

自分のターンの人が石を置いた後、相手のターンの人がどこにも石を置く事ができない（どこに置いても自分の石をひっくり返すことができない）状況になった時は、自動的に相手のターンがパスされ、自分が石を置けるマスの位置が表示されます。再度自分の石を置くマスの位置を入力してください。

## 勝敗判定

双方ともに石を置けない状態になると、ゲーム終了です。
ゲーム終了となると、最終的な盤面とその下に以下が表示されます。

```text
game set
number of Black is 黒石の数
number of White is 白石の数
winner is 石の数が多い方（Black or White）
```

それぞれの石の数が表示され、石の数が多い勝者（Black or White）が記載されます。
石の数が同じの場合は引き分けとなり、「Draw」と表示されます。
